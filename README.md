# Famous Food Store — Website

A responsive single-page website for **Famous Food Store** — Butchers, Grocers & Wholesalers in Middlesbrough.

## Contents
- `index.html` — page markup
- `css/styles.css` — styling (brand colours: royal blue, coral, white)
- `js/main.js` — mobile menu + small interactions
- `assets/` — logo and banner images

## Sections
- Hero with call-to-order
- Departments: Meat & Poultry, Groceries, Wholesale Packaging, Flour & Rice, Dairy
- About the store
- Why shop with us
- Visit / contact with embedded Google map
- Footer with social links

## Running locally
It's a static site — just open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Deploying
Works on any static host (GitHub Pages, Netlify, Vercel). For GitHub Pages,
enable Pages on this branch / `main` and point it at the repo root.
